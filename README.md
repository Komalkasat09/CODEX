The platform is designed to enhance accessibility by enabling real-time conversion between text, audio, and sign language.

✨ Key Features:
Audio/Text to Sign Language Converter
Sign Language Converter to Audio/text with autocorrect functionality
Gamified Learning Dashboard for interactive sign language education
Multiple Qiz for practice

🛠 Tech Stack:
Frontend: Next.js, ShadCN UI, Spline 3D
Backend: FastAPI with WebSocket streaming for real-time interactions
AI Integration: AI models for translation and recognition
This project stands out as it merges technology, AI, and accessibility into an innovative solution, making communication more inclusive for the deaf and hard-of-hearing community.
